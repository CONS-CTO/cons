# 霸天虎代币(CONS) - 社区自治项目

## 静态部署指南

本项目已转换为单个独立的静态HTML文件，您只需下载`index.html`文件，即可直接部署到任何静态服务器上，无需安装任何依赖。

### 部署步骤：
1. 下载`index.html`文件
2. 将该文件上传到您的静态网站服务器或托管平台
3. 访问对应的URL即可查看完整网站

### 网站功能：
- 展示霸天虎代币(CONS)的基本信息
- 社区公告和风险提示
- 图片轮播展示
- 一键复制合约地址、基金会地址和QQ群号
- 完全响应式设计，支持PC和移动设备
